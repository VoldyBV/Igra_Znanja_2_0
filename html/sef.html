Opis

Sef je logička igra. Cilj igre je otkriti kombinaciju sefa u što manje poteza.

Kako se igra?

Kombinacija se sastoji od brojeva 0, 1, 2, 3, 4 i 5. Kada napišete kombinaciju kliknite zeleno dugme (✓) da potvrdite kombinaciju. Nakon toga igra će provjeriti vašu kombinaciju i dati vam indikatore.
Postoje 3 vrste indikatora:
  • Crveni indikator - tačan broj na pravom mjestu
  • Žuti indikator - tačan broj na pogređnom mjestu
  • Sivi indikator - netačan broj

Računanje poena

Ukoliko ste pogodili kombinaciju dobijate 15 poena.
  • Ukoliko ste kombinaciju pogodili na 1. koraku, dobijate 25 bonus poena.
  • Ukoliko ste kombinaciju pogodili na 2. koraku, dobijate 20 bonus poena.
  • Ukoliko ste kombinaciju pogodili na 3. koraku, dobijate 15 bonus poena.
  • Ukoliko ste kombinaciju pogodili na 4. koraku, dobijate 10 bonus poena.
  • Ukoliko ste kombinaciju pogodili na 5. koraku, dobijate 5 bonus poena.
  • Ukoliko ste kombinaciju pogodili na 6. koraku, dobijate 0 bonus poena.

Description

The Safe is a logic game. The goal of the game is to find a code of the safe in as few moves as possible.

How to play?

The code consists of the numbers 0, 1, 2, 3, 4 and 5. Once you have written the code, click the green button (✓) to submit the code. After that the game will check your code and give you indicators.
There are 3 types of indicators:
  • Red indicator - the right number in the right place
  • Yellow indicator - right number in the wrong place
  • Gray indicator - wrong number
THE ORDER OF THE INDICATORS IS NOT THE SAME AS THE ORDER OF THE NUMBERS IN THE CODE

Points

If you guessed the code, you get 15 points.
  • If you guessed the code on the 1st move, you get 25 bonus points.
  • If you guessed the code on the 2nd move, you get 20 bonus points.
  • If you guessed the code on the 3rd move, you get 15 bonus points.
  • If you guessed the code on the 4th move, you get 10 bonus points.
  • If you guessed the code on the 5th move, you get 5 bonus points.
  • If you guessed the code on the 6th move, you get 0 bonus points.
